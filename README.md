# hexdump

Консольный hex-viewer на C++: дамп любого файла с оффсетами,
hex-парами и ASCII-колонкой.

## Возможности
- Оффсеты каждые 16 байт
- ASCII-колонка (непечатаемые — точкой)
- Проверка открытия файла, осмысленный код выхода (404)

## Известные ограничения
- Последняя неполная строка без ASCII-колонки (план на v1.1)

## Сборка
Visual Studio 2022, C++17
